Student Management System 

A Java-based Student Management System that allows administrators and students to manage student records efficiently. The application supports student registration, login authentication, attendance tracking, grade calculation, record management (CRUD operations), and role-based access for Admin and Students through a menu-driven console interface.

Key Features

Admin Login Authentication
Student Registration & Login
Add, View, Search, Update, and Delete Student Records
Attendance Percentage Calculation
Automatic Grade Generation
Pass/Fail Status Evaluation
Student Profile Management
Menu-Driven Console Interface
Data Storage using ArrayList
